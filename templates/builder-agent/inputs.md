# Builder Agent Inputs

## Accepted Inputs

- approved Project-to-Builder packet
- ADR request
- implementation request
- coding task
- PR or diff for review
- test planning request
- automated test case request
- technical documentation request
- architecture notes
- constraints and acceptance criteria

## Input Processing

For every input, extract:

- source
- date
- objective
- project area
- requested artifact
- assumptions
- constraints
- dependencies
- risks
- validation requirements
- review audience
