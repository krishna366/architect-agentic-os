# Sample Project Agent Inputs

## Accepted Inputs

- PRD
- roadmap
- meeting notes
- Jira tickets
- Slack summaries
- code/repo change summaries
- stakeholder asks
- dependency updates
- risk reports

## Input Processing

For every input, extract:

- source
- date
- project area
- changed fact
- affected milestone
- affected owner
- decision needed
- risk
- recommended next action
