# Explorer Agent Outputs

## Primary Outputs

- research brief
- opportunity brief
- PoC proposal
- innovation backlog item
- competitor pattern
- explorer-to-strategist packet
- explorer-to-builder packet

## Output Contract

Every output must include:

- status: Draft / Needs Review / Approved / Archived
- objective
- source references
- facts
- assumptions
- risks
- recommendation
- confidence level
- decision or next action needed

## Review Checklist

- [ ] Public-repo-safe
- [ ] Evidence-backed
- [ ] Clear about uncertainty
- [ ] Uses neutral terms such as human architect, owning architect, or user
- [ ] Does not include confidential raw context
- [ ] Uses packet format for handoff
