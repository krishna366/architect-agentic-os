# Navigator Agent Outputs

## Primary Outputs

- career weekly review
- career monthly review
- role evaluation
- skill-gap analysis
- achievement log
- career decision memo

## Output Contract

Every output must include:

- status: Draft / Needs Review / Approved / Archived
- objective
- sanitized facts
- assumptions
- career signals
- risks
- recommendations
- confidence level
- next actions

## Review Checklist

- [ ] No confidential project details
- [ ] No employer-specific secrets
- [ ] Portable professional language
- [ ] Facts and assumptions separated
- [ ] Recommendations include trade-offs
- [ ] Human decision remains explicit
