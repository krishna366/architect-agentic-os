# Career Signal Model

Use this model to convert sanitized work and learning inputs into portable career evidence.

## Signal Fields

- Date:
- Source:
- Sanitized Fact:
- Career Area:
- Demonstrated Skill:
- Impact Theme:
- Evidence Strength:
- Reusable Story:
- Growth Area:
- Recommended Next Action:
- Confidence Level:

## Evidence Strength

- Low: anecdotal or unverified
- Medium: credible but incomplete
- High: specific, repeated, or outcome-backed

## Impact Themes

- Reduced risk
- Improved clarity
- Increased delivery confidence
- Improved technical quality
- Accelerated decision-making
- Raised engineering standards
- Expanded influence
- Created strategic option

## Confidentiality Filter

Before storing a signal, remove:

- employer names when not needed
- customer names
- internal project names
- roadmap details
- source code
- private messages
- credentials
