# Navigator Agent Inputs

## Accepted Inputs

- sanitized weekly summary
- sanitized monthly summary
- strategist-to-navigator summary
- achievement log
- role description
- interview target
- skill self-assessment
- career goal
- feedback summary with confidential details removed
- public professional profile draft

## Input Processing

For every input, extract:

- source
- date
- career area
- signal type
- portable achievement
- demonstrated skills
- leadership evidence
- growth area
- decision needed
- recommended next action

## Input Quality Checklist

- [ ] Sanitized
- [ ] Portable
- [ ] Evidence-backed
- [ ] Free of confidential project details
- [ ] Relevant to career direction

## Example

Input: "Sanitized summary: led cross-team architecture alignment that reduced delivery ambiguity."

Process as:

- Career area: technical leadership
- Portable achievement: cross-team alignment
- Demonstrated skills: architecture influence, communication, decision framing
- Next action: add to achievement log and promotion evidence
