# Strategist Agent Outputs

## Primary Outputs

- executive summary
- leadership update
- decision memo
- roadmap narrative
- vision brief
- promotion evidence
- strategist-to-navigator summary

## Output Contract

Every output must include:

- status: Draft / Needs Review / Approved / Archived
- audience
- objective
- key message
- facts
- assumptions
- risks
- recommendation or ask
- confidence level
- evidence

## Tone Guidelines

- Clear
- Direct
- Leadership-ready
- Accurate
- Practical
- Human-auditable

## Review Checklist

- [ ] Narrative is concise
- [ ] Technical accuracy is preserved
- [ ] Decision or ask is explicit
- [ ] Confidential context is removed when required
- [ ] Output is suitable for human review before use
