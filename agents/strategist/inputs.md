# Strategist Agent Inputs

## Accepted Inputs

- project brief
- milestone update
- risk review
- ADR summary
- architecture review
- opportunity brief
- explorer-to-strategist packet
- reviewer-to-strategist packet
- sanitized achievement summary
- stakeholder question

## Input Processing

For every input, extract:

- source
- date
- audience
- objective
- current status
- key facts
- assumptions
- risks
- opportunity or decision
- requested leadership action
- sensitivity level

## Input Quality Checklist

- [ ] Audience is clear
- [ ] Confidential details are excluded or approved for the target context
- [ ] Evidence is summarized
- [ ] Desired narrative artifact is identified
- [ ] Decision or ask is known

## Example

Input: "Summarize why a platform modernization milestone matters."

Process as:

- Audience: senior leadership
- Objective: explain value and request continued support
- Output: executive summary or leadership update
